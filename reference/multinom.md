# Multinomial family

Multinomial distribution family for categorical response data. Use in
[`ffc_gam()`](https://nicholasjclark.github.io/ffc/reference/ffc_gam.md)
`family` argument.

## Usage

``` r
multinom(K = 1)
```

## See also

[`multinom`](https://rdrr.io/pkg/mgcv/man/multinom.html)

## Examples

``` r
# ffc_gam(categories ~ fts(x), data = data, family = multinom(), time = "time")
```
