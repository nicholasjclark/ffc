# Extract model.frame from a fitted `ffc_gam` object

Extract model.frame from a fitted `ffc_gam` object

## Usage

``` r
# S3 method for class 'ffc_gam'
model.frame(formula, ...)
```

## Arguments

- formula:

  a model [`formula`](https://rdrr.io/r/stats/formula.html) or
  [`terms`](https://rdrr.io/r/stats/terms.html) object or an R object.

- ...:

  Ignored

## Value

A `matrix` containing the fitted model frame
