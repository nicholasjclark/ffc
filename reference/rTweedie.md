# Random Tweedie generation

Generates random numbers from Tweedie distribution. Utility function for
Tweedie family.

## Usage

``` r
rTweedie(mu, p = 1.5, phi = 1)
```

## See also

[`rTweedie`](https://rdrr.io/pkg/mgcv/man/rTweedie.html),
[`tw()`](https://nicholasjclark.github.io/ffc/reference/tw.md)
